Where the World Drinks Safely — SDG 6.1.1

This project involved creating a global choropleth map illustrating access to safely managed drinking water using the SDG 6.1.1 indicator. The workflow began in QGIS, where I joined national-level water access data to a country boundary shapefile, cleaned attribute fields, and applied graduated color classifications.

The interactive version was built using Leaflet.js, incorporating custom styling, dynamic pop-ups, and an interactive legend to display global inequalities cleanly.

Valencia Remote Sensing Project Overview

This section displays multi-temporal satellite data analysis tracking urban development signatures and environmental indexes across Valencia, Spain, spanning from 2017 to 2024. Scroll through the analytical models below to see individual layer extractions and explicit structural breakdowns.

1. True Color Imagery (2017)

This image provides a true color (RGB) baseline snapshot of the Valencia metropolitan area in 2017. It captures urban footprint densities, coastal margins, and the surrounding agricultural greenbelts before major multi-year developments took place.

View Full Screen ↗

2. True Color Imagery (2024)

Representing the target comparative true color profile, this 2024 layout tracks recent expansion footprints. Comparing this view directly with the 2017 baseline reveals infrastructural progress and changes along urban margins.

View Full Screen ↗

3. Baseline NDVI Spectrum Analysis (2017)

This Normalized Difference Vegetation Index (NDVI) mapping visualizes photosynthetic activity intensity in 2017. Darker regions map deep canopy cover or dense cropland growth profiles, establishing environmental metrics for comparison.

View Full Screen ↗

4. Comparative NDVI Spectrum Analysis (2024)

The updated 2024 NDVI map isolates modern canopy distributions and agricultural trends. It highlights seasonal biomass variations and shows how urban developments have affected natural vegetation cover over time.

View Full Screen ↗

5. NDVI Environmental Change Delta Map

This change detection map subtracts the 2017 NDVI values from the 2024 data to calculate net biomass variations. It highlights key environmental trends, showing areas of vegetation loss alongside regions with improved green space management.
