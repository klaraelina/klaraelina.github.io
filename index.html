Where the World Drinks Safely — SDG 6.1.1

This project involved creating a global choropleth map illustrating access to safely managed drinking water using the SDG 6.1.1 indicator. The workflow began in QGIS, where I joined national-level water access data to a country boundary shapefile, cleaned attribute fields, and applied graduated color classifications.

The interactive version was built using Leaflet.js, incorporating custom styling, dynamic pop-ups, and an interactive legend to display global inequalities cleanly.

Kindergarten Accessibility in Utrecht by Bicycle

An intricate spatial vector analysis calculating commuter thresholds (ranging from 4–24 minutes) across the municipality of Utrecht. Focuses on infrastructure accessibility modeling and residential data clusters mapping out regional proximity profiles.

Flood Risk & Spatial Vulnerability Analysis

This spatial project model showcases topographic exposure and localized catchment risks. Built by evaluating hydrological overlay constraints, digital elevation structures, and flood plains configuration boundaries to evaluate vulnerabilities across environmental assets.
