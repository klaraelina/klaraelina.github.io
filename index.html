Where the World Drinks Safely — SDG 6.1.1

This project involved creating a global choropleth map illustrating access to safely managed drinking water using the SDG 6.1.1 indicator. The workflow began in QGIS, where I joined national-level water access data to a country boundary shapefile, cleaned attribute fields, and applied a graduated color classification to highlight spatial disparities. After validating the symbology and attribute structure, the dataset was exported as GeoJSON for use in a web-mapping environment. The interactive version of the map was built using Leaflet.js, incorporating custom styling, pop-ups, and a legend to support interpretation. The purpose of this project is to visualize global inequalities in water security and demonstrate how GIS can support sustainability analysis and communication.
